# Reset-SNMPAgent

## SYNOPSIS
Reset SNMP Agent to Installation Defaults

## SYNTAX

```
Reset-SNMPAgent [<CommonParameters>]
```

## DESCRIPTION
Resets the Windows SNMP Agent to Installation Default settings.

## EXAMPLES

### Example 1
```powershell
PS C:\> Reset-SNMPAgent
```

Resets the Windows SNMP Agent to Installation Default settings.
