# Remove-SNMPCommunity

## SYNOPSIS
Remove SNMP Community Names

## SYNTAX

```
Remove-SNMPCommunity -Community <String[]> [<CommonParameters>]
```

## DESCRIPTION
Remove SNMP Community Names

## EXAMPLES

### Example 1
```
PS C:\> Remove-SNMPCommunity -Community Monitoring
```

Removes the SNMP Community Name "Monitoring"
